# Desktop-Organizer
PowerShell Script to move all files on desktop into folders based on file extension. The script will check to see if the following folders exist, and create them if they do not:
User\Desktop\Documents
User\Desktop\Presentations
User\Desktop\Spreadsheets
User\Desktop\PDFs
User\Desktop\Pictures
User\Desktop\Videos
User\Desktop\Music
User\Desktop\Shortcuts

Currently supported file formats:
.docx
.doc
.pptx
.pptm
.xlsx
.xls
.pdf
.xps
.png
.jpg
.jpeg
.gif
.webm
.mkv
.mov
.mp4
.mp3
.wav
.lnk
.urv
